Game Development / RaceMania

RaceMania

// BROWSER GAME · UNITY WebGL · PLAYABLE

A one week GameJam project — built a browser-playable racing game in Unity and exported to WebGL — running entirely in your browser, no install required. Use the loader below to play, or launch it fullscreen. The HUD has parameter controls of a self-driven racer.
Or connect a controller to play. Find the controls in the bottom right legacy UI box.
The game is extrapolated using a foundation of an open-source project on GitHub. As a request from the developer here is a link to the repository.

View on GitHub ↗

— AI was used to generate the soundtrack and optimize the vehicle mechanics.
— The vehicle is models by hand in Blender.
— The track is designed by Rarui. Public assets used for game-modding and prototyping.
— Sound FX are from personal vault of the Unity Asset Store.
— Particles are made with Unity PS for exaustion and wheel smoke.
