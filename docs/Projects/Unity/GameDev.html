Overliver — Serious Game

Developed in collaboration with GameLab Oost (Enschede) and a faculty of MST. Intended for patients with NAFLD (Non-alcoholic Fatty Liver Disease) to promote healthy meal-prep habits. My scope: interactive UI elements matching the Lead Artist's style guide, recipe book system design, C# scripted functions, and Unity UI architecture.

The recipe book was a feature I proposed during brainstorms with the team and stakeholders. Written in C# with Unity UI canvas groups, it allows patients to browse recipes, mark favourites, and track meal-prep completion — all supporting the intended therapeutic habit loop.
